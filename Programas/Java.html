Java

Como o próprio nome pode sugeri o Java Script é uma linguagem de script ( é um comando a ser seguido por um software). A característica básica de uma linguagem de script, é que seu código é executado dentro de outro programa. A característica principal do Java Script é que ele pode ser executado sem a necessidade deste script passar pelo servidor, controlando o navegador, realizando comunicação assíncrona e alterando o conteúdo do documento exibido.
